      57             :  * \brief Computes the time derivative for a DG time step.
      58             :  *
      59             :  * Computes the volume fluxes, the divergence of the fluxes and all additional
      60             :  * interior contributions to the time derivatives (both nonconservative products
      61             :  * and source terms). The internal mortar data is also computed.
      62             :  *
      63             :  * The general first-order hyperbolic evolution equation solved for conservative
      64             :  * systems is:
      65             :  *
      66             :  * \f{align*}{
      67             :  * \frac{\partial u_\alpha}{\partial \hat{t}}
      68             :  *  + \partial_{i}
      69             :  *   \left(F^i_\alpha - v^i_g u_\alpha\right)
      70             :  *   = S_\alpha-u_\alpha\partial_i v^i_g,
      71             :  * \f}
      72             :  *
      73             :  * where \f$F^i_{\alpha}\f$ are the fluxes when the mesh isn't moving,
      74             :  * \f$v^i_g\f$ is the velocity of the mesh, \f$u_{\alpha}\f$ are the evolved
      75             :  * variables, \f$S_{\alpha}\f$ are the source terms, and \f$\hat{t}\f$ is the
      76             :  * time in the logical frame. For evolution equations that do not have any
      77             :  * fluxes and only nonconservative products we evolve:
      78             :  *
      79             :  * \f{align*}{
      80             :  * \frac{\partial u_\alpha}{\partial \hat{t}}
      81             :  *   +\left(B^i_{\alpha\beta}-v^i_g \delta_{\alpha\beta}
      82             :  *   \right)\partial_{i}u_\beta = S_\alpha.
      83             :  * \f}
      84             :  *
      85             :  * Finally, for equations with both conservative terms and nonconservative
      86             :  * products we use:
      87             :  *
      88             :  * \f{align*}{
      89             :  * \frac{\partial u_\alpha}{\partial \hat{t}}
      90             :  *   + \partial_{i}
      91             :  *   \left(F^i_\alpha - v^i_g u_\alpha\right)
      92             :  *   +B^i_{\alpha\beta}\partial_{i}u_\beta
      93             :  *   = S_\alpha-u_\alpha\partial_i v^i_g,
      94             :  * \f}
      95             :  *
      96             :  * where \f$B^i_{\alpha\beta}\f$ is the matrix for the nonconservative products.
      97             :  *
      98             :  * ### Volume Terms
      99             :  *
     100             :  * The mesh velocity is added to the flux automatically if the mesh is moving.
     101             :  * That is,
     102             :  *
     103             :  * \f{align*}{
     104             :  *  F^i_{\alpha}\to F^i_{\alpha}-v^i_{g} u_{\alpha}
     105             :  * \f}
     106             :  *
     107             :  * The source terms are also altered automatically by adding:
     108             :  *
     109             :  * \f{align*}{
     110             :  *  -u_\alpha \partial_i v^i_g,
     111             :  * \f}
     112             :  *
     113             :  * For systems with equations that only contain nonconservative products, the
     114             :  * following mesh velocity is automatically added to the time derivative:
     115             :  *
     116             :  * \f{align*}{
     117             :  *  v^i_g \partial_i u_\alpha,
     118             :  * \f}
     119             :  *

     121             :  * sign arises because we add it to the time derivative.
